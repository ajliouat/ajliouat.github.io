## Featured Projects
### LLMRec: RAG-Based Recommendation System 
[Repository](https://github.com/ajliouat/LLMRec) 
LLMRec is an advanced recommendation system that utilizes the Phi-3 Mini-4K-Instruct model and retrieval-augmented generation (RAG) techniques to provide personalized and context-aware product recommendations. The system iteratively refines its recommendations based on user feedback, ensuring a dynamic and adaptive user experience.

### Mistral7B Constrained Fine-tuning
[Repository](https://github.com/ajliouat/mistral7B-constrained-fine-tuning)
This project focuses on fine-tuning the Mistral7b language model with integrated rules and conditions, enabling the generation of responses that adhere to predefined constraints and guidelines. This approach enhances the model's ability to produce controlled and context-appropriate outputs.

### Machine Translation with Transformers
[Repository](https://github.com/ajliouat/machine-translation-with-transformers)
This project implements a French to English machine translation model using the Transformer architecture. The model leverages state-of-the-art techniques in neural machine translation to achieve high-quality translations between the two languages.

### Attention Is All You Need
[Repository](https://github.com/ajliouat/attention-is-all-you-need-repro)
This repository contains an implementation of the Transformer architecture, as described in the influential paper "Attention Is All You Need" by Vaswani et al. (2017). The project reproduces the key components and techniques outlined in the paper, showcasing the effectiveness of attention mechanisms in natural language processing tasks.

### Machine Vision Industrial Object Detection
[Repository](https://github.com/ajliouat/machine-vision-industrial-object-detection)
This project focuses on developing an object detection system specifically designed for industrial images. By employing advanced deep learning architectures and techniques, the system achieves high accuracy and efficiency in detecting and localizing objects within industrial settings, contributing to improved safety, quality control, and automation in manufacturing processes.
