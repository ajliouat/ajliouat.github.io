## Featured Project

### LLMRec: RAG-Based Recommendation System 

[Repository](https://github.com/ajliouat/LLMRec) | [Live Demo](https://ajliouat.github.io/LLMRec)

This state-of-the-art recommendation system harnesses the power of the Phi-3 Mini-4K-Instruct model and retrieval-augmented generation (RAG) techniques to deliver highly personalized and context-aware product recommendations. The system captures the user's evolving intent through an iterative process, continuously refining its recommendations for an adaptive and engaging user experience.

Key Features:
- Iterative and evolving recommendations based on user feedback and interactions
- Efficient inference optimizations using quantization, hardware acceleration, and embedding indexing
- Containerized deployment with Docker for scalability and portability

Technologies Used:
- Phi-3 Mini-4K-Instruct model
- ONNX Runtime, FAISS
- Docker
- Node.js
